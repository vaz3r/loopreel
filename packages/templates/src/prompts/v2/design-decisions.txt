Rules:
- Select template that best matches the content type and brand style
- Use brand colors but adapt them for the template
- Each slide should have a different layout for visual variety
- Hook slides should use large emphasis
- Value slides should use medium emphasis
- CTA slides should use large emphasis
- Background types:
  - solid: Use for clean, minimal designs
  - gradient: Use for modern, vibrant designs
  - pattern: Use for textured, organic designs
  - image: Use for content-relevant photography
- For image backgrounds, provide relevant search keywords
- Use blur (0.2-0.5) for image backgrounds to ensure text readability
- Use overlay (rgba with 0.3-0.6 alpha) for contrast on image backgrounds
- Shapes should complement the template style
- Vary layouts across slides to maintain visual interest
